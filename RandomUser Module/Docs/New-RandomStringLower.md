# New-RandomStringLower

## SYNOPSIS

Creates a lowercase random string

## SYNTAX

```
New-RandomStringLower [[-NumberOfChars] <Int32>] [<CommonParameters>]
```

## DESCRIPTION

Creates a lowercase random string 

## EXAMPLES

### Example 1

```powershell
PS C:\> New-RandomStringLower
```
